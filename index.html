                App notes

                • Static build.Everything is client-side. No backend, routing, build tools, or paid hosting required.
                • Image generation depends on model support.If your chosen image model fails, use the local SVG blueprint button or try a different OpenRouter image-capable model.
                • Best workflow.Generate blueprint → save image → import to Procreate → lower opacity → trace on a new layer → customize.
